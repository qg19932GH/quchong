# 视频去重工具 - 界面设计与淡蓝色系规范 (design_guide.md)

本文件规定视频去重工具的 GUI 视觉风格、交互规范与动效设计标准。

## 1. 视觉设计系统

本软件要求使用**极简淡蓝色系**风格，营造清爽、专业且温和的视觉体验，避免沉重和刺眼的强对比。

### 1.1 调色板 (Color Palette)

*   **背景色 (Background)**
    *   `--bg-color`: `#f0f7ff` (非常淡的亮天蓝色，呈现清爽底色)
*   **卡片/面板底色 (Card Surface)**
    *   `--card-bg`: `#ffffff` (纯白卡片，用于与淡蓝背景拉开层次)
*   **主色调 (Primary / Brand Indigo & Blue)**
    *   `--primary`: `#3b82f6` (冰蓝色/亮蓝色，用于主按钮、进度条及高亮 badge)
    *   `--primary-hover`: `#2563eb` (深冰蓝，用于悬停状态)
    *   `--primary-light`: `#eff6ff` (极淡蓝色，用于高亮背景或气泡)
    *   `--primary-text`: `#1d4ed8` (深蓝，用于浅蓝背景上的文字)
*   **危险色 (Danger / Red)**
    *   `--danger`: `#f87171` (淡红/粉红，用于删除、取消操作，避免强刺眼的大红)
    *   `--danger-hover`: `#ef4444` (中红，悬浮反馈)
    *   `--danger-light`: `#fef2f2` (粉粉红，提示背景)
*   **文字颜色 (Typography)**
    *   `--text-main`: `#1e293b` (深石板灰，保持高可读性，避免使用纯黑 `#000`)
    *   `--text-secondary`: `#475569` (中石板灰，用于副标题、元数据等)
    *   `--text-muted`: `#94a3b8` (淡灰蓝，用于失效信息或次要路径)
*   **边框颜色 (Border)**
    *   `--border-color`: `#dbeafe` (极淡的蓝色边框，代替灰边框，保持整体色彩的柔和纯净)

### 1.2 阴影与圆角 (Shadows & Radius)
*   **圆角**：
    *   大卡片、拖拽区域圆角：`border-radius: 16px`
    *   按钮、小卡片圆角：`border-radius: 12px`
    *   输入框、小图标圆角：`border-radius: 8px`
*   **阴影**（采用柔和散射阴影，增加立体感）：
    *   `box-shadow: 0 4px 20px -2px rgba(59, 130, 246, 0.05), 0 2px 6px -1px rgba(0, 0, 0, 0.02)`

## 2. 界面布局结构 (Layout)

系统高度自适应 100vh，无垂直溢出产生的双滚动条：

1.  **Header (页头)**
    *   左侧排列应用图标和标题，右侧可显示统计信息。
    *   高 70px，固定顶部。
2.  **Main Panel (主体)**
    *   上半部：拖拽文件夹区域。点击或拖入文件夹即可自动高亮显示所选路径。
    *   下半部：查重聚类列表。
    *   使用 flex-column 垂直自适应。列表部分设置 `overflow-y: auto`。
3.  **Footer (页脚)**
    *   高度 36px，展示版本号以及操作安全提示。

## 3. 核心交互动效 (Animations)

为保证用户体验，我们添加以下微动效：

*   **拖拽悬停**：当拖拽文件夹进入拖拽区时，拖拽区背景平滑过渡为 `--primary-light`，边框线变为虚线蓝色，并产生轻微向内的缩放或跳动效果。
*   **列表过滤搜索**：输入框输入字符时，卡片采用淡入淡出（Opacity & Translation）方式被过滤，避免生硬地直接消失。
*   **卡片移除与一键清理**：
    *   当点击“删除单个文件”并确认后，对应的行以 `height: 0; opacity: 0; padding: 0;` 动效收缩折叠，随后在 DOM 中删除。
    *   若重复组内只剩下最后一个文件（即不满足重复定义时），整个大卡片产生淡出收缩效果，平滑从列表中移除。
*   **呼吸指示器**：扫描时进度条顶部的 Badge 采用 `pulse` 呼吸动画，向用户展示系统正在安全运行。
